# Source Archive

This directory is the home for retired source and source-like data kept for
reference. Active source outside `SRC/ARCHIVE/` should be code/data used to
create current onboard R-YORS images or board-ingested data.

Current in-use STR8-N, HIMON V, and ASM-F2 files stay in their active source
lanes until a deliberate replacement exists. Retired samples, tests, proofs,
demo apps, helper scripts, and generated board-data fixtures should move here
in small documented batches.

Current shelves:

```text
himon/   retired HIMON/Himonia monitor branches
```

Planned shelves are tracked in
`DOC/GUIDES/PLANNING/HISTORICAL_CODE_MIGRATION_PLAN.md`.
